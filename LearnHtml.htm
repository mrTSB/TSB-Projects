Saving Your Code and Creating Your Website

Here are some quick steps on how to create your html website.

First open the notepad application on your computer. This is available for Windows 8 and 10.

Once you have notepad up you can type all of the code (which you will learn later) in there.

Once you are done typing the code select the file button then the save as button.

Next,you must name your file. At the end of the name you have to add .htm or .html. Once you do this change the text documents to all files.

Then change the coding to UTF-8. This is the preffered html encoding. The final result will look like this...

When this is done hit the save button.

WARNING: You can only view your website on your pc, creating a url will come later in the lesson.

Opening Your Code/Website

To open your website first find it in your file browser.

When you find it right click on the file and hover over the open with button. Select whatever browser you prefer to use.

This will open your website on whatever browser you selected.

However, if you want to edit your website you must select "Choose another app" and select Notepad.

This will open the notepad with all your code and will allow you to edit it.

Make sure to do Ctrl+S before you exit notepad to save your code. To see changes on your browser reload the page.

Formatting Text

You can change the text's apperance in many simple ways. All you have to do is add tags around it.

Here are some formats you can use on text. The formatting will be displayed on the word "text"

The tag < b > makes the text bold.

The tag < strong > makes the text look important.

The tag < i > italices the text.

The tag < em > emphazises the text.

The tag < mark > highlights the text.

The tag < del > is used to write text with strikthrough.

The tag < ins > is used to insert text but it really just underlines the text.

The tag < sub > puts the text in subscript

The tag < sup > puts the text in superscript.

The tag < small > makes the text smaller.

Quotations and Citations

To quote something you must use the < q > tag. This will make sure the browser includes quotes around it.

The < blockquote > tag defines a section that is quoted from another source. Most browsers indent text inside the < blockquote > tag.

Lists and Bullet Points

To create an unordered list with bullet points you must use the < ul > tag.

To created a numbered(ordered) list you must use the < ol > tag. To reverse the numbered list, you give the ol a reversed attribute so it is < ol reversed > and < /ol >. If you want to make the ordered list have letters instead of number you start with the < ol type="a" > tag and end with < /ol >

To create the actual items on the list you must use the < li > tag around each item in the list and it must be inside the < ul > or < ol > tag.
